# Projeto Spacefy - Spec Driven Development (SDD)

Bem-vindo ao repositório principal do projeto. Este repositório contém os entregáveis desenvolvidos em aula, com foco na aplicação prática da metodologia **SDD (Spec Driven Development)** e outras práticas estudadas ao longo do curso.

## 📂 Estrutura do Repositório

O repositório está organizado em duas pastas principais:

### 1. `spacefy/` (Projeto Real)
Esta é a pasta que contém o **projeto real, oficial e funcional** do Spacefy (a "Rede Social de Cifras Sincronizadas"). Toda a aplicação baseada em Next.js, com seus pacotes e configurações verdadeiras, está implementada nela.

> **⚠️ IMPORTANTE: Documentação Completa**
> **Todas as instruções detalhadas** do projeto estão localizadas dentro desta pasta. 
> Acesse os arquivos abaixo para conferir os requisitos da entrega:
> - [README.md - Como configurar, rodar dependências e exemplos de uso](./spacefy/README.md)
> - [README-Specs.md - Reflexão sobre o uso de Specs, desafios e benefícios](./spacefy/README-Specs.md)

### 2. `Project_Frondend_v0dev/` (Projeto Espelho v0.dev)
Esta pasta contém o projeto gerado inicialmente através da inteligência artificial [v0.dev](https://v0.dev/). O código presente aqui é utilizado como um **espelho ou referência visual** de como a interface deveria ficar após nós alimentarmos o sistema com as Especificações (Specs). Ele serve como um mockup interativo em código que guiou o desenvolvimento do projeto real na pasta `spacefy/`.
